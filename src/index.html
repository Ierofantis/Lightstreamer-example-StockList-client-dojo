LIGHTSTREAMER :: DOJO TOOLKIT :: STOCK-LIST DEMO
Lightstreamer
The Dojo Toolkit

StockList

A demonstration of integrating Lightstreamer and the Dojo Toolkit 1.8

This demo uses the dGrid and DojoX Charting to display the events generated by the Lightstreamer Server in both textual and graphical forms, and a special LightstreamerStore (based on dojo.store) to broker communication between Lightstreamer and the Dojo Toolkit widgets.

Use the check boxes to add and remove stocks from the chart; click any of the toggle buttons at the top to change the chart type.

Click on the grid headers to change the sort; the grid is automatically rearranged if required by a change in the sorted values.
